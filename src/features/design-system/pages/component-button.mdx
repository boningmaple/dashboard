## Log

- [x] Variants: `default`, `elevated`, `filled`, `tonal`, `outlined`, and `text`.
- [x] Sizes: `xs`, `sm` (default), and `md` (easy to extend).
- [x] Shape: default is `rounded-full`.
- [x] States: `enabled` (default), `disabled`, `hovered`, `focused`, `pressed`.
- [x] Toggle: `default`, `unselected`, `selected`, default is `default`.
- [ ] Ripple effect: todo...

> [!NOTE]
> Other colors can be applied to these button variants as long as the container and text have at least a 3:1 contrast ratio.
>  
> For example: secondary/on-secondary, error/on-error, success/on-success, etc.
